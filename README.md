# Personal Website Portfolio

## Project Overview
This personal website serves as a portfolio that highlights my journey in software development and technology. The site is designed to showcase my skills, projects, and ongoing learning while providing a professional online presence.

The website includes multiple sections such as an introduction, project portfolio, resume overview, and a space for sharing insights about my learning experiences. It is built using modern web design practices to ensure a clean layout and user-friendly navigation.

---

## Features

- **Home Page:**  
  Introduces who I am and provides a brief overview of my goals in software development.

- **Portfolio Section:**  
  Displays projects I have worked on, including web development assignments and programming projects.

- **About Me Section:**  
  Provides background information, interests in technology, and career goals.

- **Resume Section:**  
  Highlights my education, technical skills, and relevant experience.

- **Blog / Learning Section:**  
  Shares insights from my learning journey, including programming concepts and challenges I have overcome.

- **Navigation Menu:**  
  Allows users to easily move between different sections of the website.

---

## Technologies Used

- **HTML5** – Structures the content of the website  
- **CSS3** – Styles and layouts the website  
- **JavaScript (optional)** – Adds interactivity if applicable  

---

## File Structure

- `index.html` – Main homepage of the website  
- `styles.css` – External stylesheet for design and layout  
- Additional files may include project pages or assets (images, scripts, etc.)

---

## How to Run the Project

1. Download or clone the project files  
2. Open the folder in Visual Studio Code  
3. Open `index.html` in a web browser  
   - Double-click the file  
   - Or use Live Server in VS Code

## Purpose of the Project

The purpose of this website is to demonstrate my understanding of web development fundamentals, including structuring content with HTML, styling with CSS, and organizing a multi-section website. It also serves as a professional portfolio that can be shared with potential employers.
